# Webflow Collections Explained for WordPress Developers

"Do you know Webflow"?

This is a question I've been getting asked more and more often. So I experimented with it – first by using Webflow's MCP server to manipulate one the existing sites on my Webflow plan. With Claude Code I was able to do a suprisingly good job of maniuplating a site and even migrating an existing NextJS website into Webflow (this one).

But while that's all great, I thought there might be a large number of WordPress developers that are being asked the same thing, and wondering how to start.

So, I dug in, and began learning it. And in the process, I figured it'd be a good chance for me to showcase what I've learned in a video. So that's what I did. 

Here's a video on translating the concepts of Custom Post Types and ACF Field Groups into Webflow's Collections.
